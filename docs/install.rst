Installation
============
```console
# Clone the repository
git clone https://github.com/vatsal220/games.git

# Install using pip
pip3 install .
```

Usage
=====
```console
# Play Rock Paper Scissors
rps

# Play Tic Tac Toe
tictactoe

# Play Connect 4
connect4

# Play Sudoku
sudoku

# Play Snake
snake
```
