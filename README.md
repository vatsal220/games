# Games

This is a python package dedicated to playing games

## Installation
```python
# Clone the repository
git clone https://github.com/vatsal220/games.git

# Install using pip
pip3 install .
```

## Usage
```console
# Play Rock Paper Scissors
rps --config /path/to/config/config.yml

# Play Tic Tac Toe
tictactoe
```
